# EKIP Hackathon Submission Checklist

## Required artifacts
- [ ] Public GitHub repository URL
- [ ] Working deployed app URL (Azure Container Apps)
- [ ] 2-minute demo video URL
- [ ] Architecture diagram image (`architecture.png` or final version)
- [ ] Team information
- [ ] Project description (problem, solution, business value)

## Microsoft stack requirements
- [ ] Azure AI Foundry used
- [ ] Microsoft Agent Framework used
- [ ] Azure MCP server/tool integration used
- [ ] Azure OpenAI used (chat + embeddings)
- [ ] Azure AI Search used
- [ ] Azure Service Bus used
- [ ] Azure Blob Storage used
- [ ] Azure Cosmos DB used
- [ ] Azure Key Vault used
- [ ] Azure Monitor / App Insights used
- [ ] Azure AD RBAC used

## Evidence to capture
- [ ] Screenshot of upload -> queued -> processed flow
- [ ] Screenshot of citations in chat response
- [ ] Screenshot of dashboard/history view
- [ ] Screenshot of Azure resources in subscription
- [ ] CI pipeline passing (GitHub Actions)
- [ ] Deployment workflow run output
